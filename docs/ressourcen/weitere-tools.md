# Weitere Tools im Ueberblick

Diese Dienste sind fuer Forschende relevant, haben aber bewusst keine eigene
Anleitung — der Kurzueberblick reicht zum Einordnen. Details auf den
verlinkten offiziellen Seiten.

- **Research Rabbit** — Literatur-Discovery ueber Zitationsnetze, kostenlos.
  Aehnlich [Connected Papers](../werkzeuge/finden/connected-papers.md).
  <https://www.researchrabbit.ai>
- **scite.ai** — zeigt, *wie* eine Arbeit zitiert wird (bestaetigend vs.
  widersprechend). Kostenpflichtig. <https://scite.ai>
- **Consensus** — zieht direkte Antworten auf Forschungsfragen aus Papers.
  Ueberschneidet sich mit [Elicit](../werkzeuge/finden/elicit.md).
  <https://consensus.app>
- **SciSpace** — Chat mit PDFs und Literaturerklaerungen.
  <https://scispace.com>

!!! tip "Weniger ist mehr"
    Du brauchst nicht alle Tools. Such dir pro Aufgabe eines aus und bleib
    dabei — das spart Einarbeitung.
